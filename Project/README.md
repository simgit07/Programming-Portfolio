#Password Verification Program

### Description
This program simulates a simple password security check.

The program:
1. Prompts the user to enter a password
2. Checks that the password is **at least 9 characters long**
3. If valid, asks the user to correctly identify **three randomly selected characters** from the password
4. Exits immediately if any answer is incorrect
5. Displays a success message if all checks pass
