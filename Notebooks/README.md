# Fundamentals of Computer Programming – Programming Portfolio

This repository contains my solutions to the **weekly programming tasks (Weeks 3–8)** for the *Fundamentals of Computer Programming* module.  

##  Weekly Task Overview

###  Week 3 – Variables, Input, and Basic Control
Programs include:
- Greeting program with default input handling
- Password confirmation with length and common-password checks
- Looping until valid password entry
- Times tables (fixed, user-defined, and reverse output)

**Key concepts:**
- Variables and data types
- `if / else`
- `while` and `for` loops
- String handling
- Lists

---

###  Week 4 – Functions and Data Processing
Programs include:
- Input validation functions
- Uppercase/lowercase letter counting
- String manipulation
- Temperature conversion (Celsius ↔ Fahrenheit)
- Statistical processing (min, max, mean)

**Key concepts:**
- User-defined functions
- Return values
- String methods
- Lists and numerical processing

---

###  Week 5 – Scripts and Command-Line Arguments
Programs include:
- Detecting operating system platform
- Processing command-line arguments
- Finding shortest argument
- Checking website availability via HTTP response
- Command-line temperature statistics
- File backup utility

**Key concepts:**
- `sys.argv`
- Standard library usage
- File copying
- Error handling
- Command-line execution

---

###  Week 6 – Algorithms and Problem Solving
Programs include:
- Integer to binary conversion
- Factor and prime number detection
- Simple encryption and decryption
- Message obfuscation using random intervals

**Key concepts:**
- Mathematical algorithms
- Loops and conditionals
- Random number generation
- String indexing and slicing

---

###  Week 7 – Sets and Dictionaries
Programs include:
- Extracting unique letters from strings
- Set-based word comparison
- Country–capital lookup system
- Frequency analysis of encrypted text

**Key concepts:**
- Sets and set operations
- Dictionaries
- Sorting data
- Case-insensitive processing

---

###  Week 8 – File Processing and Unix Tool Reimplementation
Programs include:
- `nl` (number lines in a file)
- `diff` (compare file contents)
- `grep` (search for patterns in files)
- `wc` (line and character counting)
- `spell` (basic spell checker)

**Key concepts:**
- File input/output
- Text processing
- Command-line tools
- String searching
- Data validation
